# Safe Health Assistant

Safe Health Assistant is a team-based frontend web application developed to provide general and safe health guidance. The project focuses on helping users understand common symptoms, diet, exercise, and basic home-care practices without prescribing medicine dosage or medical treatment.

## Project Overview
This team project is designed using a rule-based approach where users can interact with a health assistant through a chat interface. The system collects basic user details such as age, gender, and known conditions to provide personalized yet safe suggestions. Emergency symptoms are identified and users are immediately advised to seek professional medical help.

## Team Contribution
This project was developed collaboratively by a team.  
Each member contributed to different parts of the application including:
- User interface design and layout
- Chat logic and rule-based response handling
- Health guidance content and safety rules
- Testing and final integration

## Key Features
- User profile creation (Name, Age, Gender, Conditions)
- Symptom-based health guidance
- Generic medicine suggestions (no dosage provided)
- One-day safe home-care plan
- Diet and exercise recommendations
- Emergency symptom detection and alerts
- Doctor consultation message template
- Chat history stored using browser LocalStorage
- English and Hindi language support

## Technologies Used
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Browser LocalStorage  

## How to Run the Project
1. Download or clone the repository  
2. Open the `index.html` file in any modern web browser  
3. No backend server or external installation is required  

## Dataset Information
This project does not use any external dataset. All responses are generated using predefined rule-based logic and static knowledge embedded directly in the JavaScript code. User data is stored locally in the browser using LocalStorage.

## Disclaimer
This application provides general health information only and does not replace professional medical advice. It does not prescribe medicines or dosages. Always consult a qualified doctor or healthcare professional for medical concerns.
